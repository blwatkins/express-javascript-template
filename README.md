# Express JavaScript Template

A template project for using [Express](https://expressjs.com/) with [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) and [webpack](https://webpack.js.org/) in the [Node.js](https://nodejs.org/en) runtime environment.

## Getting Started

Create a new project from this template using the green "Use this template" button above, then clone the new repository to your local machine.

### Installation

```shell
npm install
```

`npm install` will install all the dependencies required to run this project.
